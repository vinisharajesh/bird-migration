About

This system is to help the Ornithologist.

The project aims to Track birds that are prone to be endangered or critically populated species. As of now we focus on tracking bird migration patterns .The main aim of this system is to encourage crowd sourcing and lower commercial cost for ornithologists and research purpose.The data set for this case study comes from the LifeWatch INBO project. We process the latitude and longitude map it using python pre-defined library cartopy which results in simulated tracking graph. Later we use web application as user interface and act as a tracking analyser.
